How Council Values Are Decided

Last updated: 27 December 2025

Council Values are generated through a transparent, community-driven process designed to reflect real market sentiment while reducing bias and extreme outliers.

Important Safeguards

  • MP CAT administrators (the site owners) are not permitted to submit value suggestions and do not contribute valuation data.
  • Council Values are only assigned when sufficient data is available and the item meets eligibility requirements.
  • Council membership is limited to one representative per clan to ensure fair and balanced input.

Eligibility Requirements

An item may be considered for a Council Value only if:

  • It has at least 3 council member suggestions.
  • After removing outliers, a minimum of 3 valid suggestions remain.
  • The resulting value range appears reasonable based on observed market activity.

If these conditions are not met, the item remains unvalued until more data becomes available.

How Suggestions Work

  • Council members submit minimum and maximum value suggestions for items.
  • Each suggestion contributes to the calculation of the mean range.
  • Outliers are automatically flagged and excluded to avoid skewing results.

Calculation of Council Values

  • The mean of all valid suggestions determines the suggested range.
  • Min and max values form the final suggested range for the item.
  • A confidence indicator reflects the number of suggestions used.

How Value Ranges Are Displayed

To keep Council Values readable and useful across items of vastly different value ranges, displayed values are rounded in a consistent and predictable way.

  • Smaller values retain finer detail
  • Larger values are rounded to broader increments
  • Rounding is applied consistently across all items
  • Raw, unrounded values remain available in the Insights section for transparency

Assigning a Council Value

  • Administrators assign the final Council Value based on valid suggestions.
  • Included suggestions are linked internally to the assigned Council Value.
  • The assigned value appears on the item profile with raw numbers and metadata.

Insights

Each item with a Council Value includes an insights section showing:

  • Council Value Assigned Date
  • Council Suggestions Used
  • Confidence in Value (Low: 3–5 suggestions, Medium: 6–9 suggestions, High: 10+ suggestions)
  • Raw Minimum and Maximum Means
